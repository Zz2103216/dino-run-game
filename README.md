# 🦖🔫 武装小恐龙跑酷

一款像素风格的横版跑酷射击游戏，浏览器即开即玩！

## 🎮 玩法

- 控制小恐龙躲避仙人掌、击杀俯冲的老鹰
- 鼠标瞄准 + 射击，击杀老鹰获得护盾
- 每 50 分触发 BOSS 战，弹幕难度递增
- 地面拾取弹药箱补充子弹（无上限）

## 🕹️ 操作

| 操作 | 按键 |
|------|------|
| 跳跃 | 空格 / ↑ |
| 匍匐躲避 | ↓ |
| 瞄准 | 鼠标移动 |
| 射击 | 鼠标点击 / J / Enter |

## 🚀 部署

这是一个纯静态 HTML 文件，可以部署到任何静态托管平台：

- **GitHub Pages**：直接推送即可
- **Vercel / Netlify**：拖拽上传
- **任意服务器**：放到 web 目录下

## 💰 广告变现

1. 注册 [Google AdSense](https://adsense.google.com)
2. 获得发布商 ID（`pub-xxxxxxxxxxxxxxxx`）
3. 创建广告单元，获取广告位 ID
4. 修改 `index.html` 中的 `AD_CONFIG` 配置
5. 将 `AD_CONFIG.enabled` 改为 `true`
